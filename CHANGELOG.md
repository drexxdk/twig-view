# Changelog

All notable changes to this project will be documented in this file.

The format is based on Keep a Changelog and the project follows Semantic Versioning as closely as practical during the `0.x` phase.

## [0.1.2] - 2026-04-25

### Changed

- Refined connector rendering and animation behavior for nested branches.
- Reworked vertical spacing around items and branches to use `itemLayout.gap` more consistently.
- Improved default hover and disabled-state behavior for labels and actions.
- Expanded the demo with deeper nesting and click logging examples.

### Documentation

- Strengthened package metadata and README positioning for npm and GitHub visitors.
- Added this changelog and a contributor guide.
